## CRUD App

## Description
This is a simple CRUD web application built using Python Flask.

## Features
- Create records
- Read records
- Update records
- Delete records

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML, CSS
- No JavaScript logic used

## How to run
1. Install Flask:
   pip install flask

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## Run with Docker
1. Build the Docker image:
   docker build -t my-flask-app .

2. Run the container:
   docker run -p 5000:5000 my-flask-app

3. Open in browser:
   http://localhost:5000
